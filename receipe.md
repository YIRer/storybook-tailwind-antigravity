# storybook + tailwind + react + typescript + shardcn + radix

storybook + tailwind + react + typescript + shardcn + radix 환경을 구축해줘
이 프로젝트는 공용 컴포넌트를 만드는 프로젝트야

## 요구사항

1. 위에 적힌 스펙대로 환경을 구축 할걸
2. 공용 컴포넌트를 만드는 프로젝트이므로 npm으로 설치하여 다른 곳에서도 작동 되는 걸 상정해서 빌드 설정을 만들 것
3. storybook 환경에서 로컬로 띄웠을 때 실시간으로 수정이 가능할것
4. storybook 빌드 후 배포된 환경에서도 잘 동작 할 것
5. 중요 커밋마다 git commit을 만들 것
6. 현재 폴더에서만 파일을 생성할 것
7. 다른 프로젝트에서 tailwind 설정이 잘 동작하고 테마를 사용할 수 있도록 할 것
8. 필요하다면 마이그레이션 가이드를 만들 것
10. shardcn, radix 라이브러리를 사용하여 구현할 것
11. 테마의 경우 필요한경우 다른 프로젝트에서 언제든지 특정 값을 변경할 수 있을 것
12. Global 모달 예제하나를 만들어줄 것
